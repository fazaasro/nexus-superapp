# 2026-02-10 — Google Calendar Integration

## Task
Add Google Calendar tools to OpenClaw for scheduling and event management.

---

## Solution Found: gog Skill

**gog** — Google OAuth CLI tool that enables OpenClaw to access:
- ✅ Gmail — Read, search, draft, send emails
- ✅ Google Calendar — Check schedule, find availability, create events
- ✅ Google Drive — Search files, read documents, organize
- ✅ Google Docs/Sheets — Query content, add rows, get summaries

---

## Setup Required

### Step 1: Install gog
```bash
clawdhub install gog
# OR
npm install -g gog
```

### Step 2: Google Cloud Credentials
1. Go to console.cloud.google.com
2. Create project
3. Enable APIs: Gmail, Calendar, Drive
4. Create OAuth 2.0 credentials (Desktop app)
5. Download credentials JSON

### Step 3: Authenticate
```bash
gog auth
```

### Step 4: Test
```bash
gog calendar today
gog calendar week
```

---

## Calendar Commands Available

| Command | Purpose |
|---------|---------|
| `gog calendar today` | Show today's events |
| `gog calendar week` | Show this week |
| `gog calendar list` | List upcoming |
| `gog calendar find "term"` | Search events |
| `gog calendar create` | Create new event |
